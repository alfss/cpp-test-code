## Как собрать по Linux

### Cmake
Нужен для генерации Makefile и установки путей характерных для платформы

В каталоге с кодом:
```
cmake .
```

### make
Нужен для сборки, генерирует бинарник: journal_version_23

```
make
#чтоб очистить сборку
make clean
```
